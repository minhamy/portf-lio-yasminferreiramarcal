Projeto 1: Portfólio Pessoal

Este foi um projeto de aprendizado com foco em HTML e CSS. A ideia era criar uma página de apresentação pessoal e um portfólio simples, explorando a organização de conteúdo, um layout responsivo e a estilização de elementos básicos. O projeto demonstrou meu entendimento sobre a estruturação semântica do HTML e o poder do CSS para construir um site funcional e atraente.

Projeto 2: Blog Pessoal

Outro projeto que desenvolvi para aprimorar minhas habilidades em design e usabilidade. O objetivo era criar um blog pessoal com diferentes posts, um sistema de navegação simples e um layout minimalista. Este trabalho me ajudou a entender a importância de uma boa tipografia, espaçamento e a hierarquia visual para guiar a leitura do usuário, além de reforçar o conceito de páginas interligadas.

Projeto 3: Aprendizado de Tabela/Formulário Básico

Este projeto foi desenvolvido como parte de uma atividade prática na disciplina de Fundamentos da Programação Web da UNINTER. O objetivo principal foi criar uma página web que explorasse a funcionalidade de tabelas e formulários básicos em HTML e CSS. O trabalho permitiu colocar em prática a estruturação de dados em formato de tabela e a implementação de formulários para coleta de informações, reforçando o conhecimento sobre a organização e a interação de conteúdo em sites.
